# Spreading Love – 2D Stencyl Game

## Overview

Spreading Love is a 2D game developed using Stencyl as part of a Computer Graphics course project. The player controls an angel character whose mission is to defeat demons falling from the sky and survive through multiple levels.

## Game Concept

The gameplay is based on a simple arcade-style mechanic where the player shoots enemies while avoiding being overwhelmed. The game becomes more challenging as the player progresses through the levels.

## Features

* 2D gameplay mechanics
* Two levels with increasing difficulty
* Shooting system
* Enemy spawning mechanics
* Basic score and progression system
* Character animations and collision detection

## Technologies Used

* Stencyl
* Basic game design principles
* Computer Graphics concepts

## Project Description

This project was created as part of a Computer Graphics course and focuses on fundamental game development concepts such as sprite handling, animations, collision detection, enemy behavior, and level design. The game combines simple mechanics with progressive difficulty to create an engaging gameplay experience.
